Restaurant Website

Project Overview

This project is a restaurant website that showcases the restaurant's menu, location, and contact information. Customers can browse the menu, make reservations, and get directions to the restaurant. The website is built using HTML, CSS, JavaScript, and includes a mobile-responsive design for easy access on various devices.


Key Features

  • Menu presentation with prices
  • Reservation system with date and time selection
  • Contact information and location map
  • Responsive design for mobile and desktop
