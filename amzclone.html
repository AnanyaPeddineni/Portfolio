Amazon Clone Website

Project Overview

This project is an Amazon clone website that replicates the user interface and functionality of the Amazon e-commerce platform. It allows users to browse products, add items to their cart, and complete purchases. The website is built using HTML, CSS, JavaScript, and integrates with a backend for user authentication and order processing.


Key Features

  • Browsing and searching for products
  • Adding/removing items to/from the cart
  • User authentication and account management
  • Order placement and tracking
